# Các Future chính
- Nút Switch chuyển qua lại Dark Theme Mode - Light Theme Mode
- Nhận diện System Mode ( Dark - Light ), chuyển đổi giao diện phù hợp
- Show dialog để xác nhận khi chuyển bằng button và cancle lại lệnh.
